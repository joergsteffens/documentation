=== Notations

Replace names shown in '<angle brackets>' with the correct absolute or full path. 

Examples: 
 '<opsi-depot-share>' is the location of the opsi packages file share located at the `/var/lib/opsi/depot` path on your server.   

Likewise, the `<opsi-depot-share>/ooffice` software packet refers to the `/var/lib/opsi/depot/ooffice` path. 

Example snippets from program code or configuration files use Courier font with highlight:

[source,prompt]
----
depoturl=smb://smbhost/sharename/path
----
